# test-angular

A simple test of Angular.

## Prerequisite

| Tools          | Version |
|----------------|---------|
| Docker         | <=20.10 |
| Docker-Compose | <=1.29  |

## Installation

```bash
git clone git@github.com:BigChicChicken/test-angular.git
```

## Usage

Run application :
```batch
./start.sh
```

You have access to the application by:
[http://localhost:4200/](http://localhost:4200/)
